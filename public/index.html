📚 My Research

My current ongoing research aims to prove the usage and benifits of using CRDTs in a streaming environment, specifically when performing global aggregation functions. In traditional streaming systems, performing these functions often creates bottlenecks due to the need for synchronization which is done by applying blocking functions.

The project will consist of implementing a system capable of simulating a high capacity streaming service, with the ability to use different types of CRDTs to perform aggregations. Formal proof will be added to ensure the CRDTs provide the correct guarantees and an improvememnt in throughput and latency can be determined.

If this project sounds interesting feel free to reach out for more information 😃
